
  ip2domain：IP反查域名，查询备案信息、查询百度权重

 (由于接口对请求频率有限制，故查询速率较慢，请耐心等待)

usage: ip2domain.py [-h] [-t TARGET] [-f FILE] [-s DELAY] [-T TIMEOUT] [-r RANK] [-o OUTPUT] [--icp]

options:
  -h, --help            show this help message and exit
  -t TARGET, --target TARGET
                        Target ip/domain
  -f FILE, --file FILE  Target ip/domain file
  -s DELAY, --delay DELAY
                        Request delay (default 2s)
  -T TIMEOUT, --Timeout TIMEOUT
                        Request timeout (default 3s)
  -r RANK, --rank RANK  Show baiduRank size (default 0)
  -o OUTPUT, --output OUTPUT
                        output file (default ./output/ip2domain_{fileName}_{date}.csv)
  --icp                 With search icp (default false)
